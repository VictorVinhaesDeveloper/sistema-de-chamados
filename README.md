## ✨ O que o projeto faz

Este projeto foi pensado para facilitar a comunicação e o acompanhamento de chamados entre diferentes setores de uma empresa ou equipe.

### Principais funcionalidades

- ✅ Criação e gestão de tickets/chamados
- ✅ Dashboard com tickets abertos, em andamento e finalizados
- ✅ Chat privado por ticket em tempo real
- ✅ Controle de setores e usuários
- ✅ Perfil com avatar e nome exibido
- ✅ Atualizações em tempo real via WebSocket
- ✅ Estrutura preparada para expansão e integração

---

## 🧩 Tecnologias utilizadas

- Python
- Django
- Django Channels
- Redis
- SQLite
- HTML / CSS / JavaScript
- Pillow para upload de imagens

---

## 🏗️ Arquitetura do projeto

O sistema é dividido em módulos principais:

- `accounts` → autenticação, usuários e perfis
- `tickets` → criação e gerenciamento dos chamados
- `chat` → salas de conversa e mensagens em tempo real
- `sectors` → organização por setor
- `core` → dashboard principal

---

## 🎯 Objetivo do projeto

O objetivo é oferecer uma solução simples, útil e moderna para:

- organizar chamados internos;
- melhorar a comunicação entre setores;
- acompanhar tickets com uma interface clara;
- manter o suporte em tempo real.
